・クラス詳細説明
<Sample7>
ループ処理を行い画面表示・メニュー選択・商品の注文・お会計を順に行います。
※メソッド
	private		int			main(void)			この関数を起点に処理が動きます。

<Item>
各商品の名前・値段を保持します。
	public		string		GetName(void)		商品名を取得します。
	public		int			GetPrice(void)		商品の値段を取得します。

<baseMenu>
メニュー大項目のベースクラスです。継承先のクラスで共通して使う関数・変数を
記述しています。（大項目：麺類　小項目：醤油ラーメンという認識）
	public		vector<int>	GetItems(void)		クラス内に格納されている小項目の一覧を取得する。
	public		string		GetName(void)		大項目の名前を取得する。
	protected	void		InitItems(void)		小項目の一覧を初期化する。継承先で実際の初期化を行うためここでの処理はない。

<Noodles>
baseMenuを継承したクラスです。麺類項目として扱います。
protected		void		InitItems(void)		小項目の一覧を初期化する。麺類の種類を記述する。

<Bowl>
baseMenuを継承したクラスです。丼もの項目として扱います。
protected		void		InitItems(void)		小項目の一覧を初期化する。丼ものの種類を記述する。

<Curry>
baseMenuを継承したクラスです。カレー項目として扱います。
protected		void		InitItems(void)		小項目の一覧を初期化する。カレーの種類を記述する。

<Explanation>
このテキストのことを指します。Sample7についての説明を記述しています。
